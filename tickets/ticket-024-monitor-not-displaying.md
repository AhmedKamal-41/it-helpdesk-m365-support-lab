## User Description
"My second monitor just says 'No Signal.' My main screen works fine."

## Business Impact
User is down to a single screen, reducing efficiency for spreadsheet work. Not fully blocked.

## Troubleshooting Steps
1. Confirmed the main display worked — issue isolated to the second monitor.
2. Checked the video cable at both ends; the HDMI connector was loose at the laptop dock.
3. Reseated the cable and confirmed the monitor was powered on and set to the correct input source.
4. Pressed **Win + P** and selected **Extend** (was set to "PC screen only").
5. Verified the second monitor lit up and Windows detected it under Display settings.

## Root Cause
The projection mode was set to "PC screen only" and the video cable was loose, so the second monitor received no signal.

## Resolution
Reseated the cable, set the monitor to the correct input, and switched Windows to Extend mode. Second monitor now displays correctly.

## User-Facing Final Response
> Hi James, your second monitor was set to "off" in Windows and the cable was slightly loose. I reseated it and switched you to Extend mode — both screens are working. Handy shortcut: press Windows key + P to switch display modes. — IT Help Desk

## Prevention / Follow-Up Notes
